# 🔧 Troubleshooting Guide

## Quick Diagnostic Commands

Run these first to check your environment:

```bash
# Check Python version (need 3.8+)
python3 --version

# Check Node version (need 16+)
node --version

# Check npm
npm --version

# Check if ports are free
lsof -i :8000  # Should be empty
lsof -i :5173  # Should be empty
```

## 🚨 Common Errors & Solutions

### Error 1: "command not found: python3"

**Problem:** Python not installed

**Solution:**
```bash
# Install Python via Homebrew
brew install python3

# OR download from python.org
open https://www.python.org/downloads/
```

---

### Error 2: "command not found: npm"

**Problem:** Node.js not installed

**Solution:**
```bash
# Install Node.js via Homebrew
brew install node

# OR download from nodejs.org
open https://nodejs.org/
```

---

### Error 3: "ModuleNotFoundError: No module named 'fastapi'"

**Problem:** Virtual environment not activated or dependencies not installed

**Solution:**
```bash
cd backend
source venv/bin/activate
pip install -r requirements.txt
```

---

### Error 16: "React is not defined"

**Problem:** React import missing

**Solution:**

Check `src/App.jsx` has at top:
```javascript
import { useState, useEffect } from 'react';
```

Check `src/main.jsx` has:
```javascript
import React from 'react'
import ReactDOM from 'react-dom/client'
```

---

### Error 17: Data not showing after clicking "Analyze Profile"

**Problem:** API call failing silently

**Solution:**

1. **Open browser console (F12)**
   - Look for red errors
   - Check Network tab for failed requests

2. **Check backend logs for errors**

3. **Verify profile data format:**
```javascript
// In browser console
console.log(profile)
// Should show: {age: 25, gender: "Male", weight: 70, height: 175, goal: "General Fitness"}
```

4. **Test API manually:**
```bash
curl -X POST http://localhost:8000/api/profile \
  -H "Content-Type: application/json" \
  -d '{"age":25,"gender":"Male","weight":70,"height":175,"goal":"General Fitness"}'
```

---

### Error 18: "pandas.errors.ParserError" when loading CSV

**Problem:** CSV file corrupted or wrong format

**Solution:**

1. **Check CSV file integrity:**
```bash
cd backend
head -5 exercise_dataset.csv  # Should show headers and data
```

2. **Check for encoding issues:**
```bash
file exercise_dataset.csv  # Should show UTF-8 or ASCII
```

3. **Re-download CSV files from source if corrupted**

---

### Error 19: Slow model training (taking minutes)

**Problem:** Large dataset or slow machine

**Solution:**

1. **First run takes longer (30-60 seconds)** - this is normal
2. **Subsequent runs use cached models** - should be instant

3. **To verify models are cached:**
```bash
cd backend
ls -la models/
# Should see: rec_tree.joblib, cal_rf.joblib, meta_info.joblib
```

---

### Error 20: "npm ERR! code ERESOLVE"

**Problem:** Dependency conflict

**Solution:**
```bash
cd frontend
rm -rf node_modules package-lock.json
npm install --legacy-peer-deps
```

## 📝 Common Mistakes Checklist

- [ ] Forgot to activate virtual environment (`source venv/bin/activate`)
- [ ] Running commands from wrong directory
- [ ] Backend not started before frontend
- [ ] Ports 8000 or 5173 already in use
- [ ] CSV files not in backend directory
- [ ] App.jsx not saved after editing
- [ ] Forgot to restart servers after code changes
- [ ] Browser cache not cleared
- [ ] Wrong API_URL in App.jsx
- [ ] Missing Tailwind directives in index.css

### Error 5: "EADDRINUSE: address already in use :::8000"

**Problem:** Port 8000 already in use

**Solution:**
```bash
# Find and kill process on port 8000
lsof -ti:8000 | xargs kill -9

# Or use different port in main.py:
# Change: uvicorn.run(app, host="0.0.0.0", port=8000)
# To:     uvicorn.run(app, host="0.0.0.0", port=8001)
```

---

### Error 6: "EADDRINUSE: address already in use :::5173"

**Problem:** Port 5173 already in use

**Solution:**
```bash
# Kill process on port 5173
lsof -ti:5173 | xargs kill -9

# Or start on different port
npm run dev -- --port 3000
```

### Error 7: "Access to fetch at 'http://localhost:8000' from origin 'http://localhost:5173' has been blocked by CORS policy"

**Problem:** CORS not configured properly

**Solution:**

Check `backend/main.py` has CORS middleware:
```python
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:5173", "http://localhost:3000"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

**Restart backend after changes**

### Error 11: Tailwind classes not working

**Problem:** Tailwind not configured properly

**Solution:**

1. **Check tailwind.config.js exists:**
```bash
cd frontend
cat tailwind.config.js
```

2. **Verify src/index.css has:**
```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

3. **Reinstall Tailwind:**
```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

4. **Restart dev server:**
```bash
npm run dev
```
